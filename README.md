# 🌍 Voyago - Android Travel Intelligence

English Version Below

> **Academic Project** | MSc in Computer Engineering @ **Politecnico di Torino**
> **Course:** Mobile Applications Development (2024/2025)

Voyago è un'applicazione Android avanzata progettata per la gestione intelligente degli itinerari di viaggio.
Sviluppata come progetto d'esame, l'app si concentra sulle innovazioni stile social network e sulla UX mobile-first in scenari di connettività variabile.

---

### 🛠️ Technical Focus & Architecture

Il progetto implementa i moderni standard di sviluppo Android richiesti dal corso:

- **Language:** 100% Kotlin.
- **Architecture:** MVVM (Model-View-ViewModel) per una netta separazione delle responsabilità.
- **Persistence:** Firebase Firestore come DBMS.

# 🌍 Voyago - Android Travel Intelligence

> **Academic Project** | MSc in Computer Engineering @ **Politecnico di Torino**
> **Course:** Mobile Applications Development (2024/2025)

Voyago is an advanced Android application designed for smart travel itinerary management. 
Developed as a final exam project, the app focuses on social-network-inspired features and mobile-first UX optimized for varying connectivity scenarios.

---

### 🛠️ Technical Focus & Architecture

The project implements modern Android development standards required by the course:

- **Language:** 100% Kotlin.
- **Architecture:** MVVM (Model-View-ViewModel) for a clear separation of concerns.
- **Persistence:** Firebase Firestore as the DBMS.

---

### 🔑 Security & Historical Note
This repository reflects the entire development cycle of the course. 
Any credentials or API keys found in the commit history (dating back to the early prototyping stages) have been **deactivated and revoked** in the Google Cloud Console, along with the Cloud project itself.
